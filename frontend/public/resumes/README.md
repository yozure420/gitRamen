# Resume Images

Place exactly four image files in this directory. They will be displayed as side panels alongside the game screen.

| Filename         | Position        |
|------------------|-----------------|
| `left-top.png`   | Left panel, top |
| `left-bottom.png`| Left panel, bottom |
| `right-top.png`  | Right panel, top |
| `right-bottom.png`| Right panel, bottom |

- Accepted formats: PNG, JPG, JPEG, WEBP
- If a file is missing, that slot will be blank.
- To swap a resume, simply replace the corresponding file — no code changes needed.
